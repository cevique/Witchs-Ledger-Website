## ✨ Features

### 🌐 Showcase Website

* Clean landing page for the [**Witch’s Ledger app**](https://github.com/cevique/Witchs-Ledger)
* Sections for features, roadmap, and contact
* Smooth page navigation (no frameworks, just logic and discipline)

---

### 🎨 Umineko-Themed Design

* Gold · Red · Blue · Purple accents
* Minimal black & white fallback aesthetic
* Subtle animations + floating particles
* Theme system:

  * 🌙 Dark mode
  * ☀️ Light mode
  * ◐ System preference (default)

---

### ⚡ Interactive UI

* Scroll-triggered animations
* Page transitions without reload
* Toast notification system
* Lightweight and fast (because you’re already wasting time elsewhere)
